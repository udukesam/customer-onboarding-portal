# Customer Onboarding Portal (Demo)

This is a simple demo application simulating a customer onboarding system with mock CRM, KYC, and Payment API integrations.

## Features
- Register new clients
- Store client info via CRM mock API
- Simulate KYC verification and payment flow
- View onboarding analytics dashboard

## Run Instructions

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev

Open http://localhost:5173 to view the portal.
